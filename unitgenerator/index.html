Поля-підсилювачі

Текстовий модифікатор для одиниць, обладнання та установок; вміст залежить від конкретної імплементації.

Текстовий модифікатор для одиниць, обладнання та установок, що відображає висоту, глибину; або висоту обладнання або споруди на землі.

Текстовий модифікатор для одиниць та установок, що вказує на ефективність пристрою або можливість встановлення.

Графічний модифікатор для одиниць та обладнання, який ідентифікує напрямок руху або передбачуваний рух об’єкта.

Текстовий модифікатор для одиниць, обладнання та установок, що відображає формат DTG: DDHHMMSSZMONYYYY або "O/O" для на замовлення.

Графічний підсилювач розміщений відразу над символом. Може позначати, 1) місцевий / віддалений статус; 2) статус заручини; і 3) тип зброї.

Текстовий модифікатор для одиниць, обладнання та установок, що складається з однобуквеного рейтингу надійності та однозначного рейтингу довіри.

Текстовий модифікатор для одиниць, що вказують номер або назву команди вищого ешелону (корпус позначений римськими цифрами).

Текстовий модифікатор для обладнання; літери "ENY" позначають ворожі символи.

Текстовий модифікатор, що відображає режими та коди ідентифікації системи радіолокаційного розпізнавання.

Текстовий модифікатор для одиниць, обладнання та установок, що відображає розташування символу.

Текстовий модифікатор в символі обладнання, який ідентифікує кількість присутніх елементів.

Текстовий модифікатор у символі одиниці, який відображає (+) для посиленого, (-) для послабленого, (±) посиленого та послебленого.

Текстовий модифікатор для ворожої техніки; "!" вказує виявлені електронні підписи.

Текстовий модифікатор для одиниць; індикатор міститься всередині рамки; містить назву спеціальної С2 штаб-квартири.

Текстовий модифікатор для одиниць та обладнання, що відображає швидкість, встановлену в MIL-STD-6040.

Текстовий модифікатор для блоків, обладнання та установок; вміст залежить від конкретної імплементації.

Текстовий модифікатор для обладнання, який вказує типи обладнання.

Текстовий модифікатор для одиниць, обладнання та установок, що унікально ідентифікує певний символ або номер доріжки. Визначає номер придбання при використанні з символікою SIGINT.
